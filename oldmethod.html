WLED Software Installation

(inoffizieller Installer mit einem erweiterten Funktionsumfang)
Diese Software ist open source und wird kostenfrei zur Verfügung gestellt. Es werden keine bestimmten Eigenschaften der Software zugesichert. Der Inhaber dieser Webseite haftet nur für Vorsatz und grobe Fahrlässigkeit. Gewährleistungspflichten treffen ihn nur, wenn er Mängel der Software arglistig verschwiegen hat. Bitte beachten Sie, dass als „beta“ gekennzeichnete Versionen der Software besonders wahrscheinlich noch Fehler beinhalten. Wenn ein anderes Recht als Deutsches angewendet wird, es gelten die Bestimmungen der englischen Version (bitte oben englische Sprache wählen).

Vorgehensweise:
  1. Schliessen Sie Ihr ESP8266 / ESP32 Board an USB an.
  2. Wählen Sie den Board-Typ/Software Version aus
  3. Klicken Sie auf "Installieren" und wählen dann den korrekten COM-Port aus. Kein Board wurde gefunden?
  4. Klicken Sie auf "Install WLED". Die Installation beginnt. Sollte nach einiger Zeit die Installation beim "Wrapping Up" stehen bleiben, können es es unterbrechen. Die SW ist dann trotzdem i.O. installiert.
  5. Wichtige Hinweise zu neuen Controller-Typen: ESP32-C3, ESP32-S2, ESP32-S3 etc.

Achtung! Die ausgewählte Software wurde automatisch von einem Open Source Projekt kompiliert und durch den Betreiber dieser Webseite nicht auf die Funktionalität oder Inhalt geprüft. Es ist auch theoretisch möglich, dass diese Software schadhafte Anteile beinhaltet. Bitte installieren oder herunterladen Sie diese Software nur dann, wenn Sie der Quelle der Software vertrauen und die Risiken Ihnen bewusst sind. Die Quellen sind für Standard Versionen: https://github.com/Aircoookie/WLED, für Sound Reactive Versionen: https://github.com/atuline/WLED, für MonMod Versionen: https://github.com/MoonModules/WLED. Einverstanden!
